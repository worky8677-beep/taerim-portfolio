## Project Overview

The goal of this project is to create an exact clone of the portfolio site [https://sudharsan-design.framer.website/] while replacing the original English headings and body fonts with specific Korean fonts.

### Core Technologies
- **HTML5:** Semantic structure for a single-page portfolio.
- **CSS3:** Advanced styling including Flexbox, Grid, CSS Variables, and custom animations (Marquee, floating effects).
- **Vanilla JavaScript:** Client-side interactivity, smooth scrolling, and scroll-reveal animations using the `Intersection Observer` API.

### Key Constraints (From `prd1.md`)
- **No Frameworks:** Usage of React, Vue, or any other JavaScript frameworks is strictly prohibited.
- **Client-Side Only:** No backend, server, or database integration.
- **Typography:**
    - Headings (Big Shoulders, Oswald, Bebas Neue) → **Gmarket Sans**
    - Body (DM Sans, Albert Sans, Mluvka) → **Pretendard Std**
    - Fonts must be loaded via the specified CDNs.

## Building and Running

Since this is a client-side only vanilla project, no build step is required.

### Key Commands
- **Run:** Open `index.html` directly in any modern web browser.
- **Development:** Use a simple live server (e.g., VS Code's "Live Server" extension) to preview changes in real-time.
- **Linting/Testing:** No automated testing suite is currently implemented. Visual verification against the target site is the primary validation method.

## Development Conventions

1.  **Surgical Edits:** When modifying code, prioritize minimal changes that maintain the original design's integrity.
2.  **Naming:** Follow the existing BEM-like naming convention in CSS (e.g., `.project-card`, `.section-label`).
3.  **Modern CSS:** Utilize CSS Variables (`:root`) for colors and fonts to ensure consistency.
4.  **Performance:** Keep JavaScript lightweight and ensure animations are GPU-accelerated (use `transform` and `opacity`).
5.  **Interactivity:** Maintain the "Framer-like" feel with smooth transitions and subtle micro-interactions (e.g., custom cursor, scroll reveals).
